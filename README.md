# PixivDownloader

PixivDownloader is a **local batch download tool for Pixiv artwork**, built on Spring Boot 3.5.7 / Java 17.

PixivDownloader 是一款**本地 Pixiv 作品批量下载工具**，基于 Spring Boot 3.5.7 / Java 17 构建。
