Features · 特性

不止下载，是一整套 收藏管线。

漫画 · 动图 · 小说 全格式

插画 / 漫画 / 动图（Ugoira → WebP）/ 小说（TXT / HTML / EPUB + 封面 + 合订）四条独立管线，共享同一个 SQLite 历史库。

本地画廊 + 作品详情

侧栏检索 / 标签 - 作者三模式筛选 / 灯箱预览 / 系列上下章 / 相关作品 / 同作者其他作品。下完即可在本地浏览。

收藏夹归档 + 自定义模板

支持自定义图标 / 独立下载目录的收藏夹；文件名模板：{artwork_id}_p{page}、{author_name}、{R18+} 等占位符自由组合。

多人模式 · 配额 · 邀请码

solo / multi 两套模式。多人模式下每个 UUID 独立配额，超额打包归档为 ZIP；可签发限定 Tag / 作者 / 分级的邀请码，访客只能看你允许的作品。

暗色模式 + 多语言

所有面向用户的新页面走 i18n bundle（中文 / English），主题切换持久化。CSS 变量驱动，深色模式与亮色模式视觉对齐。

SQLite WAL + GUI 工具

本地库记录每件作品的 ID / 标题 / 文件夹 / 格式 / R-18 / AI / 标签。Swing GUI 自带状态面板、配置编辑、图片分类、目录有效性检查、数据回填工具。

Get Started · 开始使用

三步，把它跑起来。

Windows installer 一键安装；其余平台 Java 17 + HTTP 代理即可。

01

下载并启动

Windows 直接装 *-win-x64-setup.exe（含 GUI 管家）；其他平台从 Releases 下载 PixivDownload.jar 后 java -jar。

02

首次配置 setup

浏览器自动打开 /setup.html，选择 solo / multi 模式，设置用户名密码，配置好就能开搜。

03

直接用 pixiv-batch

访问 /pixiv-batch.html，四个 tab 任选；想要原页面下载按钮可在 🧩 油猴脚本 卡片一键安装。
