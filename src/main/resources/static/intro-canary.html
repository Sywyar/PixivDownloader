Features · 特性

不止下载，是一整套 收藏管线。

多人模式 + 配额

solo 单人鉴权 / multi 多人共享两套模式。多人模式下每个 UUID 独立配额，超额自动打包归档成 ZIP 供用户下载。

Ugoira 动图自动转码

后端下载 ZIP 帧包，ffmpeg 按每帧延时合成 WebP。不再是一堆解不开的 ZIP，而是能直接在相册里播的动图。

ImageClassifier 可视化分类

Swing 桌面端 + 远程 API 双模式。看图点选目标文件夹，后端代为移动文件 + 更新 SQLite 记录。批量归档效率拉满。

SQLite 历史数据库

WAL 模式的本地库记录每一件作品：ID、标题、文件夹、格式、R18、归档状态。重复下载自动跳过，查询与统计秒级响应。

Get Started · 开始使用

三步，把它跑起来。

前后分钟级部署，Java 17 + 代理 127.0.0.1:7890 即可。

01

下载 JAR 并启动

从 Releases 下载 PixivDownload.jar，终端运行 java -jar。首次启动会生成 config/config.yaml。

02

安装油猴脚本

Tampermonkey 导入本仓库提供的 *.user.js。单作品下载、批量导入单作品、User、Search、Page 批量五款任选。

03

访问 setup 完成配置

浏览器打开 /setup.html 选择 solo / multi 模式，填入 Pixiv Cookie，之后就可以开始收藏之旅了。
