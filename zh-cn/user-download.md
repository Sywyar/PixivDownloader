# 画师批量下载

「User 模式」通过输入画师 ID 或主页链接，一键下载该画师的全部作品（插画/漫画/动图或小说）。

---

## 使用步骤

1. 打开 `http://localhost:6999/pixiv-batch.html`
2. 切换到「**👤 User 模式**」Tab
3. 粘贴画师主页链接或数字 ID
4. 选择下载范围（插画或小说）
5. 点「**解析并预览**」
6. 配置附加筛选（可选）
7. 点「**全部入队**」或手动选择部分作品入队
8. 开始下载

---

## 支持的链接格式

直接粘贴以下任意格式，系统会自动提取数字 ID：

```
https://www.pixiv.net/users/123456
https://www.pixiv.net/users/123456/artworks
https://www.pixiv.net/users/123456/illustrations
https://www.pixiv.net/users/123456/novels
https://www.pixiv.net/users/123456/request/artworks   ← 约稿页链接（自动切到「约稿」范围）
https://www.pixiv.net/en/users/123456
123456                                       ← 只填数字 ID 也可以
```

---

## 下载范围切换

| 范围 | 包含内容 |
|------|---------|
| **插画 + 漫画 + 动图** | 该画师所有图片类作品 |
| **小说** | 该画师所有小说 |
| **约稿** | 该画师**已完成并公开的约稿（リクエスト）成品**——约稿成品本质是普通插画，按插画的预览 / 入队 / 下载 / 判重链路处理 |

预览区**一次只能获取一种类型**——切换后旧预览结果会清空。但下载队列不受影响，可以先把插画加入队列，再切换到小说获取一次、再入队，最后一并下载。

> 直接粘贴约稿页链接（`https://www.pixiv.net/users/{id}/request/artworks`）时会**自动切到「约稿」范围**，无需手动选择。

---

## 附加筛选

展开「附加筛选」卡片可以缩小范围，例如：
- 只下载该画师的 R-18 作品
- 排除 AI 生成作品
- 只下载收藏数 100 以上的作品

筛选不仅在预览时生效，**实际下载时也会对每个作品逐个过滤**，不符合的自动跳过。

---

## 存为计划任务

如果你想**定期自动补充**该画师的新作品，可以把当前配置保存为计划任务：

向下滚动到「⏰ 存为计划任务」卡片 → 填任务名 → 设触发频率 → 点「创建任务」。

任务创建后，后台会按设定的时间间隔自动发现并下载该画师的新作，你完全不需要手动操作。

详见[计划任务](/zh-cn/scheduled-tasks)。

---

## 数据量提示

一位画师可能有几百甚至几千张作品，首次下载可能需要较长时间。可以：
- 用附加筛选缩小范围
- 调低「最大并发数」减轻对 Pixiv 服务器的压力
- 设置「作品间隔」（毫秒），在作品之间等待一段时间
