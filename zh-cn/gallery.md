# 作品画廊

作品画廊是查看、整理、管理已下载作品的核心页面。

访问地址：`http://localhost:6999/pixiv-gallery.html`

---

## 界面概览

画廊页面主要分为三个区域：

- **左侧边栏（导航模块）**：视图切换（全部图片 / 按作者 / 按系列漫画）、收藏夹列表及新建按钮、导航链接（画廊 / 下载 / 监控 / 统计 / 疑似重复 / 邀请码管理）
- **中央网格**：作品缩略图列表
- **顶部工具栏**：搜索框、排序、筛选、视图切换

侧边栏右缘有两个外凸按钮：

- **导航**：展开 / 收起左侧边栏。
- **任务列表**（仅管理员）：展示后台压缩任务（批量导出、打包）的进度与剩余有效期，任务完成后可在列表中手动下载压缩包；过期任务自动清理，服务重启后列表清空。点击批量导出后会自动打开该面板。

顶部工具栏还有画廊类型切换标签：**漫画**（当前插画/漫画/动图画廊）和**小说**（跳转到小说画廊 `pixiv-novel-gallery.html`）。

---

## 搜索

搜索框支持多种搜索范围（点击搜索框左侧的下拉切换）：

| 范围 | 说明 |
|------|------|
| **综合** | 标题 + 作者名（默认） |
| **标题** | 按作品标题搜索 |
| **作者名** | 按作者名称搜索 |
| **作品 ID** | 精确匹配作品 ID |
| **作者 ID** | 精确匹配作者 ID |
| **简介** | 按作品简介搜索 |
| **标签（模糊）** | 标签包含关键词 |
| **标签（精确）** | 标签完全匹配 |

---

## 筛选

点击工具栏「**筛选**」按钮展开筛选面板：

- **内容分级**：全部 / 全年龄 / R-18 / R-18G
- **AI 生成**：全部 / 排除 AI / 仅 AI
- **图片格式**：JPG / PNG / GIF / WebP 等
- **收藏夹**：按收藏夹筛选

### 标签/作者三态筛选

这是画廊最强大的筛选功能，每个标签或作者可以单独设置为三种状态：

| 状态 | 含义 |
|------|------|
| ✅ 必须有 | 作品必须包含该标签/来自该作者 |
| ❌ 不能有 | 作品不能包含该标签/来自该作者 |
| 🔵 或者有 | 满足任意一个「或者有」条件即可 |

多个「必须有」之间取**交集**，多个「或者有」之间取**并集**，然后再排除「不能有」。

## 排序

| 排序选项 | 说明 |
|---------|------|
| 下载时间 | 按入库时间排序（默认，最新在前） |
| 作品 ID | 按 Pixiv 作品 ID 排序 |
| 图片数 | 按该作品的图片总数排序 |
| 状态 | 按作品的下载/归档状态排序 |
| 画师 ID | 按画师 ID 排序（同一画师的作品聚在一起） |
| 标签数 | 按标签数量排序 |
| 漫画系列 | 有系列的作品按系列排序聚在一起 |

每种排序均支持升序/降序切换。

## 查看作品详情

点击任意作品缩略图，进入 `pixiv-artwork.html` 详情页：

- 标题、作者、发布时间、标签
- 作品简介
- 多图浏览（支持左右切换、全屏灯箱）
- 相关作品推荐
- 该作者其他作品
- 跳转 Pixiv 原站按钮

## 批量管理（管理员）

在画廊顶部工具栏点「**批量管理**」进入多选模式，勾选作品后通过底部操作条的「**操作**」菜单执行批量动作。选择方式：

- **全选本页 / 清除选择**：仅作用于当前页。
- **全选所有结果**：选中当前筛选条件下的**全部**结果（可再点单个作品取消个别项），翻页不丢失选择。

「操作」菜单提供三种动作：

- **导出**：弹窗中可选择**打包方式**（按作者 / 作品分类，或以作品 ID 为文件夹名）、**打包格式**（当前支持 ZIP）以及**是否导出后删除源文件**。确认后在后台打包，自动打开「任务列表」面板展示进度，完成后自动开始下载，也可随时在任务列表中手动下载；压缩包内附带记录作品信息与原始路径的 `manifest.json`。勾选「导出后删除」时，**只有打包成功后**才会删除源文件与下载记录。
- **添加到收藏夹**：把选中作品批量加入指定收藏夹。
- **删除**：批量删除选中作品。

### 删除作品

> ⚠️ 删除会移除作品在本地的图片文件与派生数据（标签、收藏夹关联与缩略图缓存），文件**不可恢复**，且仅管理员可用。

- **删除单个作品**：在作品详情页点红色「**删除**」按钮，弹窗确认后删除。
- **批量删除**：批量管理多选后在「操作」菜单点「**删除**」并确认。

小说画廊提供完全相同的批量管理入口，小说阅读页提供相同的单本删除入口。

**删除标记**：被删除的作品会在下载历史中保留一个「删除标记」——它们对画廊、筛选、统计等所有浏览入口不可见，但批量下载、油猴脚本与计划任务在「跳过已下载作品」时仍能识别出「**已经下载过，但被删除**」，默认不会把它们当作新作品重新下载。如果确实想重新下载，在下载设置中勾选「**允许已删除的作品被重新下载**」即可；重新下载成功后删除标记自动清除，作品重新出现在画廊中。

## 收藏夹管理

收藏夹用于在本地对作品进行分类整理（独立于 Pixiv 收藏，仅在 PixivDownloader 内有效）。

### 新建收藏夹

左侧边栏底部点「**+ 新建收藏夹**」，填写名称即可。

### 自定义设置

右键收藏夹或进入收藏夹编辑页面：
- **重命名**
- **自定义图标**（支持 PNG / JPG / WEBP，最大 1MB）
- **自定义下载目录**（下载时自动存入该目录）
- **删除**

### 将作品加入收藏夹

在画廊缩略图上悬停，点出现的「❤️」图标；或进入详情页点「收藏到...」按钮。

## 作品赏析模式

在详情页点「**作品赏析**」按钮，进入 `pixiv-showcase.html`，沉浸式全屏浏览当前作品，无干扰阅读。

## 访客邀请

管理员可以创建邀请码，分享画廊的查看权限给其他人（仅查看，不能下载或修改）：

1. 画廊页面右上角点「**邀请访客**」
2. 设置过期时间、内容分级（可限制 SFW 范围）、标签/作者白名单
3. 复制生成的邀请链接发给对方

访客通过 `http://host:port/invite?code=xxx` 访问，只能看到白名单范围内的作品。

详细邀请管理见 `http://localhost:6999/pixiv-invite-manage.html`。
