# 快捷获取

「快捷获取」是 `pixiv-batch.html` 打开后默认显示的第一个 Tab，凭你已保存的 Pixiv Cookie 自动识别账号，一键拉取个人相关数据加入下载队列。

?> 此功能需要先保存含 `PHPSESSID` 的 Cookie。未保存时按钮显示为禁用状态并有提示。

---

## 可获取的数据

| 数据源 | 说明 |
|--------|------|
| **我的收藏** | 插画/漫画 + 小说，含「不公开」收藏 |
| **我的作品** | 你自己投稿的插画/漫画 + 小说，含「不公开」 |
| **我的约稿作品** | 你自己已完成并公开的约稿（リクエスト）成品（按插画处理） |
| **我的关注** | 所有已关注的画师列表，含「不公开关注」；支持按用户名/ID 实时过滤 |
| **已关注的用户的新作** | 你关注的所有画师的最新插画/漫画/动图（Pixiv「フォロー新着作品」），可按页翻阅 |
| **我的珍藏集** | Pixiv 珍藏集（コレクション）中的作品 |

---

## 使用步骤

1. 打开 `http://localhost:6999/pixiv-batch.html`
2. 确认默认停留在「**⚡ 快捷获取**」Tab
3. 点击对应按钮（「我的收藏」「我的作品」等）
4. 等待数据加载完成，作品列表展示在下方
5. 点「**全部入队**」或勾选单个作品后「**选中入队**」
6. 队列出现后点「**开始下载**」

---

## 关注列表的二层展开

点击关注列表中的某位画师，会在下方展开其作品预览（插画/小说可切换）。

可以：
- 点单张图片旁的「入队」加入单个作品
- 点「整页入队」把当前页所有作品都加入
- 翻页后继续入队

---

## 珍藏集

点击某个珍藏集，展开其中的插画+小说混合列表，操作方式与关注列表相同。

---

## 附加筛选（实时预览过滤）

展开任意作品列表后，上方的「**附加筛选**」卡片可按内容分级 / AI / 标签 / 作品类型 / 页数 / 字数 / 收藏数实时过滤当前预览页：

- 「整页入队」加入的是**过滤后**的作品；
- 「全部入队」仍会入队全部，不符合条件的会在实际下载时被逐个跳过并标注原因。

未展开具体作品列表（如停留在关注列表 / 珍藏集列表选择页）时，卡片会提示先展开具体作品列表。

---

## 存为计划任务（管理员）

快捷获取里的**每一种来源**都能存为后台自动下载的[计划任务](/zh-cn/scheduled-tasks)：

| 来源 | 计划任务效果 |
|------|------|
| 我的收藏（插画/小说 × 公开/不公开） | 定期把新增收藏下载下来 |
| 我的作品 | 定期下载你自己的新投稿 |
| 我的约稿作品 | 定期下载你自己新完成的约稿成品 |
| 已关注的用户的新作 | 定期下载关注画师的新作（フォロー新着） |
| 点进某位关注画师 | 定期下载该画师新作 |
| 点进某个珍藏集 | 定期下载珍藏集内新增的插画与小说 |

展开来源后，页面底部的「**⏰ 存为计划任务**」卡片会显示来源说明，填好名称与触发方式即可创建；未展开来源时会提示需先展开。

?> 收藏 / 关注新作 / 珍藏集属于账号私有内容，对应任务必须已授权含 `PHPSESSID` 的 Cookie 才能运行（Solo 模式新建时会自动绑定）。编辑这几类任务时来源为只读，更换来源请删除后重新创建。
