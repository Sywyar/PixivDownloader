# URL 批量下载

「批量导入单作品」Tab 是最灵活的下载方式：粘贴一批作品 URL 或 ID，一次性批量下载，支持插画和小说混合。

---

## 快速上手

1. 打开 `http://localhost:6999/pixiv-batch.html`
2. 切换到「**🎨 批量导入单作品**」Tab
3. 在文本框里粘贴 URL 列表
4. 点「**获取信息**」
5. 预览确认后点「**开始下载**」

---

## 输入格式说明

文本框每行一条，支持多种格式混合：

```
# 完整 URL（最常用）
https://www.pixiv.net/artworks/12345678
https://www.pixiv.net/novel/show.php?id=87654321

# 只填数字 ID（默认当插画处理）
12345678
12345679 | 作品标题（标题可选，留空自动获取）

# 区段头：指定后续 ID 的类型
artwork:
11111
22222
novel:
33333
44444 | 这是小说标题

# 兼容 OneTab、N-Tab 导出格式
https://www.pixiv.net/artworks/55555 | Tab 标题
```

### 区段头说明

`artwork:` 和 `novel:` 单独成行，大小写不限，全角/半角冒号均可。

放在区段头下方的**纯数字 ID**（`id | title` 格式）会按该类型解析，直到遇到下一个区段头或文本末尾。

完整链接（以 `https://` 开头）**始终按链接本身的类型**解析，不受区段头影响。

---

## 附加筛选

展开「**附加筛选**」卡片，可以在下载前对队列中的作品进行过滤：

| 筛选项 | 说明 |
|--------|------|
| **内容分级** | 全部 / 全年龄 / R18+ / R-18 / R-18G |
| **AI 作品** | 全部 / 排除 AI / 仅 AI |
| **标签** | 精确匹配 / 模糊匹配（逗号分隔，需全部命中） |
| **收藏数** | 最少多少收藏才下载 |
| **页数** | 插画页数范围 |
| **字数** | 小说字数范围 |
| **作品类型** | 插画 / 漫画 / 动图 |

筛选在**实际下载时**对每个作品逐个执行，不符合条件的作品会在队列中显示跳过原因（如「内容分级不符」「标签不匹配」）。

---

## 下载设置

展开「**下载设置**」卡片，可调整：

- **文件名模板**：支持 `{artwork_id}`、`{author_name}`、`{page}` 等变量，详见[配置参考](/zh-cn/configuration)
- **下载目录**（收藏夹）：下载到哪个本地收藏夹
- **自动收藏**：下载完成后自动通过 Pixiv API 收藏该作品（需要 Cookie）
- **小说格式**：TXT / HTML / EPUB

---

## 兼容 OneTab / N-Tab 导出

从 OneTab 或 N-Tab 浏览器扩展导出标签页列表后，可以直接粘贴进来，格式为 `URL | 标题`，PixivDownloader 会自动解析。

不是 Pixiv 作品链接的行（如其他网站）会被自动跳过。
