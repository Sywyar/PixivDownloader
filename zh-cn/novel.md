# 小说下载

PixivDownloader 支持下载 Pixiv 小说，格式可选 TXT / HTML / EPUB，还支持小说系列合订本。

---

## 下载单本小说

### 方式一：URL 批量导入

1. 打开 `pixiv-batch.html` → 切换到「**🎨 批量导入单作品**」Tab
2. 粘贴小说链接：
   ```
   https://www.pixiv.net/novel/show.php?id=12345678
   ```
3. 或粘贴纯 ID，前面加 `novel:` 区段头：
   ```
   novel:
   12345678
   ```
4. 选择格式（TXT / HTML / EPUB）
5. 开始下载

### 方式二：快捷获取（你自己的小说/收藏）

打开「**⚡ 快捷获取**」Tab → 点「我的收藏（小说）」或「我的作品（小说）」，直接入队。

---

## 下载小说系列

1. 打开 `pixiv-batch.html` → 切换到「**📚 系列模式**」Tab
2. 粘贴系列链接：
   ```
   https://www.pixiv.net/novel/series/123456
   ```
3. 点「**解析并预览**」，显示系列所有章节
4. 选择「**全部入队**」或部分章节入队
5. 开始下载

### 生成系列合订本（EPUB）

系列下载时可同时生成 EPUB 合订本：

- 展开下载设置，勾选「**生成系列合订本**」，选择合订格式（推荐 EPUB）
- 合订本文件保存在 `novel-series-{seriesId}/` 目录下，文件名格式为 `{系列标题}.epub`（AI 译文合订本为 `{系列标题}_{语言代码}.epub`）

EPUB 合订本特性：
- EPUB3 规范，支持多级目录（系列 → 章节）
- 内嵌系列封面图片
- 携带书籍元数据（简介、标签、来源链接）

?> 推荐使用 EPUB 格式阅读，章节目录和排版效果最好。

### 在系列目录页随时下载合订本

进入小说系列目录页（`pixiv-series.html?type=novel&seriesId=...`），右上角操作区新增「**下载合订本**」按钮：

- 点击后弹窗让你选择语言：**原文**，或该系列已有的任一译文语言；
- 选择非原文语言时弹窗会提示：合订本只会用所选语言替换**已翻译**的章节，**未翻译章节回退原文**；若希望整本完全使用所选语言，请先点击「翻译整个系列」翻译完所有章节后再下载；
- 确认后后端按当前数据库状态即时生成最新的 EPUB 合订本并触发浏览器下载，无需事先勾选「生成系列合订本」。

## 格式说明

| 格式 | 特点 |
|------|------|
| **TXT** | 纯文本，兼容性最好，无格式 |
| **HTML** | 保留原始排版（Ruby 注音、标题等），可在浏览器直接打开 |
| **EPUB** | 适合电子书阅读器（Kindle、Apple Books 等），支持目录和封面 |

## 本地阅读

下载完成后，可以在小说画廊浏览：

```
http://localhost:6999/pixiv-novel-gallery.html
```

画廊支持按标题、作者、系列、标签搜索，还支持**正文全文检索**（基于本地索引，可以搜索小说内容里的文字）。

### 在线阅读

点击小说缩略图进入 `pixiv-novel.html?id=<id>`：
- 查看正文、标签、作者、系列章节导航
- 上一章/下一章快速跳转

## 听书（TTS 朗读）

在小说阅读页点击「**🎧 听书**」按钮，逐段朗读正文。

两种语音引擎：

| 引擎 | 特点 |
|------|------|
| **浏览器引擎**（默认） | 完全离线，即时，不消耗服务器资源；音质取决于系统语音包 |
| **在线引擎（Edge 神经语音）** | 微软在线 TTS，音质自然，语言丰富；需联网（经后端代理） |

播放控制：
- 播放/暂停、上一段/下一段、停止
- 当前朗读段高亮并自动滚动
- 进度条可跳转到任意位置
- 支持选择语音和调整语速

朗读进度会自动保存，下次打开同一本小说可以从上次继续。

## AI 多角色朗读（管理员）

在小说阅读页点「**🎭 多角色朗读**」，由大模型把整章逐句归属到说话人（旁白 / 各角色），每个角色用一段固定的音色画像，逐句合成并连续播放——区别于上面的单一嗓音「听书」。需要配置大模型，以及一个支持音色描述的朗读引擎（如自行启动的 VoxCPM）。

- **逐句播放 + 跟随**：按句合成、顺序播放并预取下一句；当前句所在**段落高亮并自动滚动**，朗读条上显示「说话人：当前句」字幕。
- **旁白音色**：首次分析前的设置弹窗里可为本作旁白选定一种音色基调（温暖女声 / 沉稳男声 / 中性，可试听）；确认后该旁白音色会被**锁定**、AI 之后不再改写，避免旁白音色逐章漂移。不选（保持当前）则沿用本作现有 / 默认旁白（默认为温暖女声）。
- **选角与音色**：点朗读条上的「🎭」打开面板，查看每个角色的性别 / 年龄 / 英文音色画像；可**编辑某角色音色并保存**（保存后即标记为「已锁定」，AI 之后不再覆盖），也可单独试听。编辑音色对**后续合成立即生效**，无需重新分析。
- **角色参考音克隆**：在面板里可为任意角色（含旁白）**一键生成一段「标准音」**或**上传真人参考音**（wav/mp3，可附转录）。配了参考音后，该角色的逐句朗读会**克隆这段参考音的音色**——音色跨句 / 跨章更稳定一致，同时仍保留每句的情绪起伏；参考音可试听、删除，变更后已缓存的逐句音频会自动按新音色重算。可在配置里关闭克隆或调整克隆生成上限。
- **音色冲突**：当正文证据与某个你锁定的角色音色明显矛盾 / 不完整时，会在面板里列出冲突，可选择「采纳建议 / 保留当前 / 改写」。
- **分段字数**：与「AI 翻译」同义——`0`（默认）表示整章一次性分析（上下文最全、归属最准）；填数字则按段落累积到约该字数分多批分析（更省 token）。
- 分析结果（逐句归属）会**缓存**，重播不重算；需要重新分析时在设置里点「重新分析」。超长正文会拒绝一次性分析并提示缩小范围。

## AI 翻译（管理员）

在小说阅读页「听书」按钮左侧点「**AI 翻译**」，把正文翻译成指定语言并保存到本地，之后直接查看、无需重复翻译。

> 需先在 GUI「AI / 大模型」中配置并启用大语言模型（OpenAI 兼容协议）。

翻译弹窗：

- **目标语言**：默认按当前界面语言预填（中文界面填「简体中文」、英文界面填「english」），可自由修改填写任意语言；AI 会判断该语言是否真实存在，不存在会提示。
- **分段字数**：`0` 表示整篇一次性翻译；填一个数字（如 `5000`）则把长文按段落累积到约该字数切成多段、逐段翻译再拼接，适合超长篇。
- **已翻译过该语言时**：可选择「覆盖重译」或「跳过」。

确认开始翻译后会弹出进度弹窗：实时显示当前用时计时，底部提供「隐藏」和「取消」按钮。「隐藏」只关闭弹窗、后台继续翻译，翻译进行中再次点击「AI 翻译」按钮会重新弹出当前进度；「取消」会中断当前请求并丢弃前端结果（后端可能仍写入数据库，无法拦截）。本章翻译成功后，如果该小说属于某个系列，会自动重生该语言变体的系列合订本，与下文「翻译整个系列」入口保持一致。

### 名词映射表（保证专有名词一致）

为避免人名、地名、组织名等专有名词在不同段落 / 章节里被翻译成不同写法，翻译弹窗里多了一个「**名词映射表**」选项（在「已翻译过该语言时」下方）。

- 一张映射表里是一组「**原文 → 各语言译名**」的对应关系。默认会带出该作品的映射表：**系列里的小说共用同一张表**（默认名为系列名），**单独的小说各用一张表**（默认名为小说名）；你也可以选「不使用」、选用其它已有的表，或「＋ 新建映射表」。
- 选中映射表后，表里的译名会随翻译请求一并发给 AI，让它按既定写法翻译，保证术语一致。
- AI 在翻译中遇到表里还没有的新专有名词时，会把译名回报回来并**自动并入所选映射表**，供后续段落、后续章节以及以后重译复用——一边翻一边积累，越翻越一致。
- 点选项旁的「**编辑**」会依次打开三层弹窗：先改表名、选择或添加目标语言，再对所选语言用「**原文 ↔ 译名**」成对输入框逐条增删映射。

?> 整个系列一起翻译时各章共用同一张映射表，新发现的译名会实时累积到后续章节，使全系列术语保持统一。

### 切换内容语言

小说阅读页与系列目录页新增「**内容语言**」下拉，可在原文与已有的各译文语言之间切换查看正文。该切换**独立于界面语言**（只改正文显示、不改界面），并会被记住；「听书」也会朗读当前所选语言的译文。

切换到某种译文语言时，标题也会自动换成译后标题：

- 小说详情页：小说标题切换为译后小说标题；
- 系列目录页：顶部系列名 + 各章节卡片的标题都切换为译后标题；
- 语言变体合订本（EPUB）：系列名 + 各章节标题也使用译后版本。

某章节 / 系列名尚未翻译时回退到原文标题。

**标题翻译的上下文策略：**

- 章节标题与该章节正文在**同一次 AI 请求**里翻译（共享上下文 + 同一张名词映射表），保证标题中的人名 / 地名 / 设定术语与正文用同一种译法。
- 系列名的翻译是独立请求（没有"正文"可附），但请求里会带上：该系列在同语言下**已翻译完成的章节标题对**（原标题 → 译后标题）作为命名 / 风格样例，以及当前选用的名词映射表，让系列名跟章节标题的术语用法保持一致。

### 翻译整个系列

在系列目录页点「**翻译整个系列**」，以章节为单位逐章翻译为指定语言（弹窗与单篇一致，可选跳过已翻译章节）。完成后会为该语言生成独立的系列合订本 `{系列标题}_{语言代码}.epub`：已翻译的章节用译文，未翻译的章节仍保留原文，**原文合订本不受影响**。

整个过程显示进度弹窗：顶部给出「请勿关闭标签页 / 单篇耗时较长 / 取消行为说明」三条提示，下方展示当前章节序号、本轮成功 / 跳过 / 失败统计与当前用时。可「隐藏」让翻译在后台继续，也可「取消」终止后续章节；取消后已翻译完成的章节保留在本地数据库，并仍会触发该语言合订本生成。在「跳过已翻译」模式下，会先以一次小型 AI 请求把填写的目标语言名解析为规范语言代码（弹窗显示「正在识别语言代码」状态），随后已存在该语言译文的章节直接跳过、不再为识别语言而触发一次完整翻译。

?> 重新生成系列合订本时会一并刷新所有已存在译文语言的合订本，并替换其中已翻译的章节。

## 删除小说（管理员）

与作品画廊一致，管理员可删除已下载的小说：在小说阅读页点红色「**删除**」按钮，或在小说画廊点「**批量管理**」多选后批量删除。删除会同时移除本地的正文 / 封面等文件与数据库记录，**不可恢复**。
