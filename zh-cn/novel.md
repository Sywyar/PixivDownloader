# 小说下载

PixivDownloader 支持下载 Pixiv 小说，格式可选 TXT / HTML / EPUB，还支持小说系列合订本。

---

## 下载单本小说

### 方式一：URL 批量导入

1. 打开 `pixiv-batch.html` → 切换到「**🎨 批量导入单作品**」Tab
2. 粘贴小说链接：
   ```
   https://www.pixiv.net/novel/show.php?id=12345678
   ```
3. 或粘贴纯 ID，前面加 `novel:` 区段头：
   ```
   novel:
   12345678
   ```
4. 选择格式（TXT / HTML / EPUB）
5. 开始下载

### 方式二：快捷获取（你自己的小说/收藏）

打开「**⚡ 快捷获取**」Tab → 点「我的收藏（小说）」或「我的作品（小说）」，直接入队。

---

## 下载小说系列

1. 打开 `pixiv-batch.html` → 切换到「**📚 系列模式**」Tab
2. 粘贴系列链接：
   ```
   https://www.pixiv.net/novel/series/123456
   ```
3. 点「**解析并预览**」，显示系列所有章节
4. 选择「**全部入队**」或部分章节入队
5. 开始下载

### 生成系列合订本（EPUB）

系列下载时可同时生成 EPUB 合订本：

- 展开下载设置，勾选「**生成系列合订本**」，选择合订格式（推荐 EPUB）
- 合订本文件保存在 `novel-series-{seriesId}/` 目录下，文件名格式为 `{系列标题}.epub`（AI 译文合订本为 `{系列标题}_{语言代码}.epub`）

EPUB 合订本特性：
- EPUB3 规范，支持多级目录（系列 → 章节）
- 内嵌系列封面图片
- 携带书籍元数据（简介、标签、来源链接）

?> 推荐使用 EPUB 格式阅读，章节目录和排版效果最好。

---

## 格式说明

| 格式 | 特点 |
|------|------|
| **TXT** | 纯文本，兼容性最好，无格式 |
| **HTML** | 保留原始排版（Ruby 注音、标题等），可在浏览器直接打开 |
| **EPUB** | 适合电子书阅读器（Kindle、Apple Books 等），支持目录和封面 |

## 本地阅读

下载完成后，可以在小说画廊浏览：

```
http://localhost:6999/pixiv-novel-gallery.html
```

画廊支持按标题、作者、系列、标签搜索，还支持**正文全文检索**（基于本地索引，可以搜索小说内容里的文字）。

### 在线阅读

点击小说缩略图进入 `pixiv-novel.html?id=<id>`：
- 查看正文、标签、作者、系列章节导航
- 上一章/下一章快速跳转

---

## 听书（TTS 朗读）

在小说阅读页点击「**🎧 听书**」按钮，逐段朗读正文。

两种语音引擎：

| 引擎 | 特点 |
|------|------|
| **浏览器引擎**（默认） | 完全离线，即时，不消耗服务器资源；音质取决于系统语音包 |
| **在线引擎（Edge 神经语音）** | 微软在线 TTS，音质自然，语言丰富；需联网（经后端代理） |

播放控制：
- 播放/暂停、上一段/下一段、停止
- 当前朗读段高亮并自动滚动
- 进度条可跳转到任意位置
- 支持选择语音和调整语速

朗读进度会自动保存，下次打开同一本小说可以从上次继续。

## AI 翻译（管理员）

在小说阅读页「听书」按钮左侧点「**AI 翻译**」，把正文翻译成指定语言并保存到本地，之后直接查看、无需重复翻译。

> 需先在 GUI「AI / 大模型」中配置并启用大语言模型（OpenAI 兼容协议）。

翻译弹窗：

- **目标语言**：默认按当前界面语言预填（中文界面填「简体中文」、英文界面填「english」），可自由修改填写任意语言；AI 会判断该语言是否真实存在，不存在会提示。
- **分段字数**：`0` 表示整篇一次性翻译；填一个数字（如 `5000`）则把长文按段落累积到约该字数切成多段、逐段翻译再拼接，适合超长篇。
- **已翻译过该语言时**：可选择「覆盖重译」或「跳过」。

### 切换内容语言

小说阅读页与系列目录页新增「**内容语言**」下拉，可在原文与已有的各译文语言之间切换查看正文。该切换**独立于界面语言**（只改正文显示、不改界面），并会被记住；「听书」也会朗读当前所选语言的译文。

### 翻译整个系列

在系列目录页点「**翻译整个系列**」，以章节为单位逐章翻译为指定语言（弹窗与单篇一致，可选跳过已翻译章节）。完成后会为该语言生成独立的系列合订本 `{系列标题}_{语言代码}.epub`：已翻译的章节用译文，未翻译的章节仍保留原文，**原文合订本不受影响**。

?> 重新生成系列合订本时会一并刷新所有已存在译文语言的合订本，并替换其中已翻译的章节。

## 删除小说（管理员）

与作品画廊一致，管理员可删除已下载的小说：在小说阅读页点红色「**删除**」按钮，或在小说画廊点「**批量管理**」多选后批量删除。删除会同时移除本地的正文 / 封面等文件与数据库记录，**不可恢复**。
